Quick Start — import in seconds

  1. Install the native host once: run the installer in the NativeHost folder (install.bat on Windows, install.command on macOS, install.sh on Linux).
  2. Open the side panel (click the extension icon) and press Import from Chrome — or use the Import button in Settings.
  3. Pick which Chrome profile to pull from. Only new groups are added; existing data is never overwritten.
  4. Organize in Workspace, remove clutter in Cleanup, open any group anytime from the side panel.

Connection keeps dropping? Read this

Symptom: the first connection works, then Test Connection / Import "disconnects" — often after the Dashboard has been open for a while.

This is almost always a leftover native-host process. Quick fix:

  1. Press 📡 Test Connection above. If it succeeds, you're good — just retry your action.
  2. If it fails, open Task Manager (Ctrl+Shift+Esc) → Details tab → end any stray python.exe or pythonw.exe left by the host.
  3. Reload the extension at chrome://extensions (🔄). Then press Test Connection again.
  4. Still failing? Fully close Chrome (end every chrome.exe in Task Manager), reopen, and re-run NativeHost\install.bat.
  5. As a last check, open NativeHost\com.tabgroup.master.json and confirm allowed_origins matches the Extension ID shown above.

Your saved groups are stored inside the extension and are never affected by a host disconnect — only Import and Backup use the host.
