1-Click Automation Setup

  1. Download the NativeHost folder from GitHub to your local computer.
  2. Double-click and run the install.bat file inside the folder.
  3. The system will automatically detect your PC paths, mask your local username for privacy, and securely register the Native Host within Chrome. No manual JSON editing required!
