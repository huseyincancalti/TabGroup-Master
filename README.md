# TabGroup Master 🚀
### Ultimate Full-Screen Hierarchical Workspace & Tab Group Manager for Google Chrome

TabGroup Master is a powerful open-source extension that elevates Google Chrome's native tab grouping feature into a full-screen, nested workspace manager inspired by **VS Code** and **Unity hierarchy tree structures**. 

It is tailor-made for power users who are tired of cluttered tab management, want to organize hundreds of tabs into clean directories, and need a reliable workflow without fearing session loss.

---

## 🔥 Key Features

* **Hierarchical Folder System (Tree View):** Organize your categories and tab groups into unlimited nested folder structures. Enjoy a buttery-smooth drag-and-drop experience powered by **SortableJS**.
* **Smart Inbox Logic:** Closed or newly detected tab groups land in a dedicated "Inbox" panel first. They are safely held there as unassigned items until you drag them into your dashboard folders.
* **Global & Smart Search (Search Highlight):** Search across Active, Saved, and Conflict tabs from a single unified input bar. Features input smoothing for case and keyboard layout variations (e.g., instantly matching Turkish/English character variations like ç-c, ı-i, ş-s) and highlights matched keywords in neon yellow!
* **Advanced Session Recovery:** Even if Chrome crashes or restarts—causing live tab groups to reset with completely new group IDs—the system intelligently matches them by their titles, keeping your custom dashboard folder hierarchy perfectly intact.
* **JSON Export/Backup:** Export your entire workspace configuration into a clean JSON file with a single click. Your data never stays locked inside the extension.
* **Privacy-First Automation:** The background macro engine (`importer.exe`) runs 100% locally on your machine. During installation, it automatically masks your PC's local folder username (`C:\Users\username`) to safeguard your privacy.

## 🛠️ Windows Automated Setup Guide (Quick Start)

To enable the automated background macros (Auto-Scan) and sync seamlessly with hidden groups, follow these two quick steps:

### Step 1: Install the Chrome Extension
1. Go to the **Releases** tab of this repository, download `TabGroup_Master_Extension_v0.1.0-alpha.zip`, and extract it to a permanent folder on your PC.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Toggle the **Developer mode** switch in the top-right corner.
4. Click **Load unpacked** in the top-left corner and select the extracted extension folder.
5. **Copy the unique Extension ID** generated by Chrome for TabGroup Master.

### Step 2: Connect the Macro Engine (Native Host)
1. Go to the **Releases** tab, download `TabGroup_Master_NativeHost_v0.1.0-alpha.zip`, and extract it into a secure directory on your computer.
2. Open the `NativeHost` folder, right-click the `install.bat` file, and select **Edit** (or open it with Notepad).
3. Find the line containing `chrome-extension://...`, replace the placeholder ID with your **actual Extension ID** copied in Step 1, and save the file (make sure to leave the trailing slash `/` intact).
4. **Double-click `install.bat`** to run it.
5. The script will automatically calculate your local paths, safely register the Native Host within the Windows Registry, and finish the configuration in a split second!

## 🛡️ Why Does It Need a .exe / .bat File?

Google Chrome's strict security sandbox protects users by preventing browser extensions from reading out-of-bounds open windows or sniffing background system tabs. To bypass this restriction and trigger the automated **Auto-Scan** macro safely, this project utilizes Google's official **Native Messaging API**. 

All scripts and binaries inside the `NativeHost` directory are fully open-source, easily inspectable, and operate 100% offline. No external tracking or network requests are made.
