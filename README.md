# TabGroup Master 🚀
### Ultimate Full-Screen Hierarchical Workspace & Tab Group Manager for Google Chrome

TabGroup Master elevates Google Chrome's native tab grouping into a full-screen, nested workspace manager inspired by **VS Code** and **Unity hierarchy tree structures**.

It is tailor-made for power users who want to organize many tab groups into clean directories, back them up as JSON, and restore them later — with **no external programs, no setup, and nothing reading your computer**.

---

## 🔥 Key Features

* **Hierarchical Folder System (Tree View):** Organize tab groups into unlimited nested folders with a buttery-smooth drag-and-drop experience powered by **SortableJS**.
* **Capture Saved Groups (no native host):** Chrome only exposes groups that are open. Click **Capture Groups**, then open your saved groups from Chrome's tab-group bar one by one — each is stored instantly and closed again, so memory stays flat. Or use **Capture open groups** to snapshot everything currently open at once.
* **Smart Inbox Logic:** Captured groups land in a dedicated "Inbox" until you drag them into folders.
* **Global & Smart Search (Search Highlight):** Search across Active and Inbox groups with case- and layout-insensitive matching (Turkish/English variants like ç-c, ı-i, ş-s) and neon-yellow highlights.
* **Session Recovery:** When Chrome restarts and live groups get new IDs, they are re-matched by title so your folder hierarchy stays intact.
* **JSON Export & Import:** Export your entire workspace to a portable JSON file and import it back on any machine. Your data never stays locked inside the extension.

---

## 🛠️ Quick Start

1. Open Chrome and navigate to `chrome://extensions/`.
2. Toggle **Developer mode** in the top-right corner.
3. Click **Load unpacked** and select this extension folder.
4. Click the toolbar icon to open the side panel — that's it. No `.bat`, no `.exe`, no registry changes.
